Computer Science Course Plan

This table really isn't that interesting but I wanted to become farmilar with creating tables using html. # Name
110 Computation, Programs, and Programming This is one of the best courses I've ever taken. We spent the majority of the semester building simple algorithms primarily using recursion. Each day we would build on what we learned previously and by the end of the semester we were able to build some very interesting programs
121 Models of Computation Discrete math for computer science. It was refreshing taking a math course that wasn't entirely calculus. This course required alot of self study to do well in but I walked away fairly confident in my ability to solve simple circuits and proofs
210 Software Construction This course intoduces object oriented programming. It's nice starting to use a real languange (java) and work with a dynamic IDE (intellij). Most of this course involves analyzing and debugging small programs that have been partially completed. I really started to enjoy this course near the end of the semester when we started working on design patterns and built an android app
317 Internet Computing I'm not actually too excited to take this course but think it will be super useful to understand network infrastructure and routing before I start in the industry
319 Software Engineering Project Throughout my degree I want to get as much experience as I can working with groups on large-scale projects. Group projects seem to be as close as I can get to what Agile Development will be like
322 Introduction to Artificial Intelligence This is one of the courses I am most excited to take. AI is a super interesting field and use of AI is starting to become commonplace in various industries. Even though this is an introductory course, I think it will invaluable in the future
340 Machine Learning and Data Mining This is the course that made me decide to take computer science. I'm a big math and stats geek and the I'm eager to see they types of problems I can solve combining my mathematical background with computer science. Along with AI I think machine learning is the future for not just computer science but many other industries
